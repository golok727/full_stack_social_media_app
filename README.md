# A Social Media App
A photo sharing app made with React and Django

## Quick Start
Start `server`

```bash
cd server
python manage.py migrate # Run the migrations
python manage.py runserver
```
Start `client`

```bash
cd client
yarn dev
#or
npm run dev
```
